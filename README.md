Apna-Weather Dashboard 🌦️💻
A real-time, responsive Weather Analytics Dashboard built with PHP, optimized to provide precise metrics and handle regional micro-climates effectively (including heat index stabilization for coastal and interior regions like Karachi and Dadu).

🚀 Features
Comprehensive Pakistan Weather Updates: Instantly check critical metrics including Temperature, Humidity, Rain Forecasts, Thunderstorms, Wind Speed, and Solar Radiation.

Stabilized RealFeel Engine: Includes a custom mathematical compression layer to regulate extreme heat index overshoots caused by high humidity.

Dynamic Weather Icons: Automatically maps API weather conditions to beautiful vector icons (Clear, Cloud, Rain, Heavy Showers, and Thunderstorms).

Interactive Dark Maps: Integrated with Leaflet.js for beautiful, high-performance dark-themed live map tracking.

Geolocation Support: Built-in "Auto" button to instantly track weather based on your current browser location.

🛠️ Tech Stack
Backend: PHP (cURL API consumption & data handling)

Frontend: HTML5, CSS3 (Glassmorphic Layout, Grid System)

Mapping API: Leaflet.js (Dark Matter vector tiles)

Icons: FontAwesome 6

Weather Data Source: Visual Crossing Weather API
